# 💀 Rift's Project Graveyard

This organization is a personal archive for my **dead, abandoned, or obsolete projects**. By this, I aim to talk about projects I don't have the heart to keep on my main account and that are...

- Outdated experiments ;

Feel free to browse, but don’t expect updates or support. These projects are buried, not maintained.
